# LinkedIn Project Showcase Guide: Marketplace Negotiator Bot

Posting about this project is a great way to demonstrate your skills in **Browser Extensions**, **AI Integration (Gemini)**, and **Web Scraping**.

Here is a strategy to make your post go viral and attract recruiters/peers.

## 1. The Narrative Hook

Don't just say "I built an extension." Tell a story.
**The Story Arc:**

1. **Inspiration**: "I watched Michael Reeves' video about a scam bot..."
2. **The Challenge**: "I wanted to replicate it but use modern tech (Google Gemini 1.5) and make it accessible as a Chrome Extension."
3. **The Hurdles**: "Facebook blocked me. I had to engineer a stealth mode."
4. **The Solution**: "A 'self-healing' extension that sees what you see (Computer Vision) and negotiates for you."

## 2. Visuals (Crucial)

LinkedIn favors video content.

* **Hero Asset**: A 30-60 second screen recording.
  * *Start*: You on a Facebook listing (e.g., that Royal Enfield).
  * *Action*: Open popup -> Select "Aggressive (Reeves Style)" -> "Hindi" -> Click Generate.
  * *Climax*: Show the funny/sarcastic response appearing.
  * *End*: Show the code snippet of the "Stealth Mode" optimization.
* **Carousel**: If you can't do video, do a PDF carousel with:
  * Slide 1: Expectation vs Reality (The Bot).
  * Slide 2: How it works (Architecture Diagram).
  * Slide 3: One of the funniest generated responses.

## 3. Key Technical Skills to Highlight

Tag these in your post to hit keywords:

* `#GoogleGemini` / `#GenerativeAI` (for the brain)
* `#ChromeExtension` (for the platform)
* `#ComputerVision` (for analyzing product photos)
* `#JavaScript` / `#WebScraping` (for the core logic)
* `#StealthEngineering` (creative problem solving)
